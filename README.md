# ViralSidekick

AI creator intelligence. Paste your channel, ViralSidekick autopsies every upload against what wins in your niche: outliers, dead zones, posting windows, and a 30-day roadmap.

**Live:** https://viralsidekick.com

A [Stromation](https://www.stromation.com) product. Built in public.

## stack
Static HTML/CSS/JS (GitHub Pages) · n8n analysis engine · OpenAI · Stripe · Supabase

## v1 scope
- Free scan on public YouTube data (no OAuth): outlier forensics, dead-zone detection, posting windows, AI verdict + fixes + 30-day roadmap
- Pro $19/mo (Stripe): roadmap refresh, viral DNA, next-video generator, priority access to retention autopsy + TikTok
- Retention autopsy + TikTok: roadmap (needs YouTube OAuth verification / TikTok dev app)
